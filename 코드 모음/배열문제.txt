배열 문제

요구 사항에 맞는 프로그램을 만들어라.

사원의 [번호], [이름], [부서], [연봉] 을 저장하고 관리하는 프로그램을 만들라.

사원을 정의하는 클래스 staff, 콘솔의 기능을 담당하는 cmd, 기능을 수행하는 staffTest 
총 3개의 클래스로 나누어 만들도록 한다.

요구 사항

1. 첫 콘솔창 화면에 [사원 추가], [사원 검색], [사원 탐색], [나가기] 메뉴를 만들고
사용자의 선택에 각 기능이 수행될 수 있도록한다.

2. [사원 추가]
사원의 번호는 사원이 추가될 때마다 자동으로 1씩 증가하도록 한다.
이름과 부서는 String, 번호와 연봉은 int 형으로 한다.
추가할 때마다 새로운 인스턴스를 만들어 배열에 저장하여 관리한다.

3. [사원 검색]
사용자로부터 사원의 이름을 받아 해당 사원이 있을경우 해당 사원의 정보를
출력해준다. 해당 이름의 사원이 없을 경우 사원이 없음을 출력하고 첫 화면으로
돌아온다.

4. [나가기]
프로그램의 기능을 종료하는 기능이다.

###### 난이도 Up
 - 삭제 기능을 추가한다.
 - 사원 검색 기능에서 부서명 혹은 번호를 입력받으면 해당하는 모든 사원의 정보를 출력한다.

###### 더 난이도 Up
 - 콘솔창에 수정기능을 추가하여 사원의 이름을 탐색 후 해당 사원의 정보를 수정할 수 있도록
한다.
